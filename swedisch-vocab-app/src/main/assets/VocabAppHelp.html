Help and Information

Introduction

This app will help you learn swedish vocabulary.

Settings

There is a number of settings available.

Number of Daily Words

The number of words in the daily list of vocabs to be learn. If this setting is changed, the vocab list must be updated in order to see the change.

Word Separator

The word saparator is needed when new words are added to separate different german translation of different grammar. Example separator = ",": german1,german2,german3.

Persistence Method

This setting defines where the dictionary is loaded from and where new words are written to.

  • User File: A file which is saved locally on the device. At the beginning no words will be there. All of them have to be added manually.
  • App File: A file is read in which comes with the app. Saving user-defined words is not supported.
  • Hardcoded: For Testing only. There is a set of hardcoded words. New words are added to the list temporary but will not be saved permanently.

If this setting is changed a reload of the dictionary or a restart of the app is required.

Debug Mode

If switched on it is possible to access debug information.

Adding Words

Just fill in the form and click on add. The prefix must be selected:

  • en or ett: for nouns.
  • att: for verbs.
  • none: for any other word.

En, ett or att must not be included in what is typed in the swedish field. Note that different grammar and different german translations have to be separated by the word separator defined in the Settings.

Learning Vocabs

When using the app for the first time there is no list of words to be learned. So as a first step an update of the vocab list needs to be done.

Once that is done learning can be started. One can choose between Swedisch-German and German-Swedish. The App will memorize which words were marked as known words. When updating the vocab list the next time the known words will be replaced by new/other words. The learning progress can be reset at any time.

View Dictionary

There is the possibility to view the whole dictionary. All the words are shown in a list. Clicking on one of the words gives detailed information about the word. There is also the option to search for words. Just type the beginning of the word and the list will show all words matching that pattern.

Modifying Words

There is the possibility to modify words. When viewing the word details click on the button "Edit Word".

  • It is possible to delete a word from the whole dictionary.
  • Adding/removing the word from the new list is supported. This is useful if a certain word from the dictionary should be learned again. When it is on the new words list it will at some point end up on the to learn list.
  • Last but not least it is possible to change the word itself. This is useful to add remarks or to correct typos.

Dictionary

Reload Dictionary

This is needed to change the persistent type.

Export/Import Dictionary

There is the possibility to export the whole dictionary and import it again. Currently this is supported for persistence method "File" only. This is useful to create a backup of the words entered into the app.
